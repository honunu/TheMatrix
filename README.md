# The Matrix - Multi-Agent System

## 简介
基于《黑客帝国》角色的多智能体协作系统，用于复杂系统分析、代码开发、金融市场研究等场景。

## 环境配置
```bash
pip install -r requirements.txt
```

## 使用方法
```bash
python main.py
```

## Agent架构
- **Architect**: 系统架构者，任务分发与资源协调
- **Morpheus**: 导航者，意图理解与策略规划
- **Oracle**: 先知，风险评估与趋势预测
- **Neo**: 救世主，复杂问题决策与创新突破
- **Trinity**: 执行者，高效执行任务
- **Smith**: 防御者，异常检测与安全审计
- **Cypher**: 记忆管理者，共享上下文

## 配置
修改 `config/settings.yaml` 中的LLM配置。
